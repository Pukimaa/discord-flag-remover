# Discord Flag Remover
Automatically removes flag reactions from Discord messages, unless they are not related to a country.

## ✨ Installation
### 📥 Prerequisites
- [Bun](https://bun.sh/)

### 🔋 Install Dependencies
```bash
bun install
```

### 🌐 Start the bot
```bash
bun run start
```
